From github to git to github
1.Create Repo in github.
2.  git clone <HTTPS link>
3.After new files or changed files
    git add .
4.commit
    git commit -m <Some message>
5.Push to github
    git push origin main

From local to git to github1.
1.Create new repo => init
    go to folder -> cd ..   
                 -> mkdir <NewName>
    go to new Repo -> cd <NewName>
                   -> git init 
    create new files
    Add changes    -> git add .
    commit         -> git commit -m "Add new files"

Go to Github -> Go to Repo
    click New -> Repo Name <local Repo>

-> git remote add origin <link>
-> git remote -v (to verify remote)
-> git branch (to check branch)
-> git branch -M main (to rename branch)
-> git push origin main (git push -u origin main)
-> git add .
-> git commit -m "Add README"
-> git push (if -u is already used)
